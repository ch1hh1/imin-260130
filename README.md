# AIご案内係

社内問い合わせの一次対応を支援するフロント（一般利用者向けチャット画面）です。

## フロントの起動方法（ローカル）

フロントのみの動作確認は、以下の手順で行えます。

1. `frontend` ディレクトリに移動します。
2. 依存関係をインストールします: `npm install`
3. 開発サーバーを起動します: `npm run dev`
4. ブラウザで表示されたURL（例: http://localhost:5173）にアクセスします。

```bash
cd frontend
npm install
npm run dev
```

- 質問を入力して送信すると、約1.5秒後にモックの回答と参照元が表示されます。
- 「新規会話」ボタンで会話履歴をクリアできます。
- 本フロントはローカルPC上でのみ動作するモック版です（RAG/AI連携は未実装）。
